⚠️ Modo de Almacenamiento Local (Sin Base de Datos) Activo

No se pudo conectar a la base de datos central (Netlify PostgreSQL o Supabase). El panel ha activado el modo local (localStorage) de respaldo. Las actividades y publicaciones que guardes aquí solo se guardarán en tu navegador actual y no serán visibles para otras personas o dispositivos desde otras computadoras.

👉 Para solucionarlo, debes configurar tu base de datos en Netlify (variables de entorno) o ingresar tus credenciales de Supabase en js/config.js.
